# CiweimaoDownloader

下载刺猬猫的小说为txt和epub，避免书本下架，不能下载未购买章节。

### 本软件优点
   1. 支持 ADB 自动拉取模式，连接模拟器后自动获取文件，无需手动操作。
   2. 下载在手机端进行，软件无需登录，不接触账号相关数据。
   3. 自动生成全文txt、epub以及单章txt，且自动解析文章中的图片url后整合到epub中，且位置不变。
   4. 程序生成epub符合其3.3版本的规范。
   5. 程序使用结构体Chapters和Book传递参数，简洁易读。
   6. 程序使用了多线程和异步特性，处理速度快。
   7. 支持配置文件预设全部参数，零交互运行。

温馨提示：本软件仅为了个人存档、学习使用，想要传播盗版、损坏原作者利益的请自觉离开

### 运行方式

- **Windows**: 解压后双击 `run.bat`
- **Linux**: 解压后运行 `./run.sh`

无需安装 Python，所有依赖均已包含在 `python/` 目录中。ADB 工具也已内置在 `adb/` 目录中。

## 使用教程

详见压缩包内 `使用说明.md`

## 构建

```bash
# Windows (PowerShell)
.\package.ps1

# Linux
./package.sh
```

构建产物位于 `dist/` 目录。

## 开发

```bash
pip install -r requirements.txt
python src/main.py
```
